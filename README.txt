				   SUPER TANKS
         				            -x-

Jogo produzido para a disciplina Algoritmos e Estruturas de Dados do curso de Engenharia da Computação da Universidade Federal do Rio Grande - FURG.

Feito por João Vitor Hartung Toppa (matrícula 155456) e Pedro Roberto Fernandes Noronha (matrícula 155463).

Todos os sprites foram produzidos pelo grupo.

Para executar o jogo, clique em ARCADE.py para o modo arcade e 2PLAYER para o modo versus ou execute-o pelo VSCode apertando Ctrl+A seguido de Shift+enter.

Se o programa apresentar o erro de GraphWin não reconhecido, abra a biblioteca graphics.py pelo editor de texto do VSCode.

Bibliotecas necessárias:
keyboard.py
graphics.py

Sistema recomendado:
Windows 11/10/8

Versão do Python recomendada:
3.9.6 ou superior

Controles:

Jogador 1:
WASD - Movimentação
Z - Tiro

Jogador 2:
Setinhas - Movimentação
N - Tiro
